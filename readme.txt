== Description ==
A simple ads widget that uses the native WordPress media manager to add ads widgets to your site.

== Installation ==
1. Upload the entire ks-ads-widget folder to the /wp-content/plugins/ directory
2. Activate the plugin through the \"Plugins\" menu in WordPress
3. In your Widgets menu, simply drag the widget labeled \"Ks Ads Widget\" into a widgetized Area.
4. Enjoy it!!!
